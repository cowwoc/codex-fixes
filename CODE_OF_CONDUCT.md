# Code of Conduct

## Our Standards

Participants are expected to:

- communicate professionally and respectfully
- keep reports focused on actionable technical details
- avoid harassment, abuse, insults, threats, or discriminatory behavior
- respect maintainer time and repository policies

## Scope

This applies to issue discussions, release discussions, and other repository interactions.

## Enforcement

The maintainer may remove comments, lock conversations, or block participants for conduct that interferes with productive maintenance of the repository.

Because this repository does not accept pull requests and has a funding-gated support model, repeated attempts to bypass repository policy may also result in moderation action.
