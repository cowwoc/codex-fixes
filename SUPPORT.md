# Support Policy

This repository does not provide general support for Codex.

## Maintainer Response Policy

The maintainer only replies to:

- issues with active issue funding
- serious funding inquiries
- quote requests
- professional-services requests
- issues explicitly marked `maintainer-approved`

Unfunded issues may be closed automatically without further discussion.
